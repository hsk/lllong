object compact

Compactパーサ

Compactは意味を持たないC言語のような式言語です。 compactオブジェクトを使う事でCompactをパースできます。

下降型の演算子順位法を用いて構文解析します。

usage:

val exp = compact("1+2*3")
println(exp)
